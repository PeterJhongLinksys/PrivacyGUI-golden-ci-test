# PrivacyGUI Golden CI (test)

Orchestration repo that drives golden (screenshot) testing of
[linksys/PrivacyGUI](https://github.com/linksys/PrivacyGUI) on a schedule.

**Start here:** read [`IMPLEMENTATION_PLAN.md`](./IMPLEMENTATION_PLAN.md). It contains the
full design rationale (sections 1–9) and a self-contained, task-by-task implementation plan.

This repo is intentionally empty except for the plan — open Claude Code inside this repo and
have it build out the workflows/scripts per the plan.
